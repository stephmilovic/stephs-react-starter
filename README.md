# Steph's React Starter


## Features

* React 16
* Webpack 4
* Babel
* CSS Modules
* ESLint
* Staging ES Next Features
* Hot Module Replacement

## Installation

* `git clone git@github.com:stephmilovic/stephs-react-starter.git`
* cd stephs-react-starter
* npm install
* npm start
* visit `http://localhost:8081/`
